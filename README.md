# Spotify Music Recommendation System (Clustering Project)

This project performs clustering on Spotify audio features such as:
- Danceability
- Energy
- Loudness
- Acousticness
- Tempo

Machine Learning Model Used:
- **K-Means Clustering**

Features Included:
- Correlation matrix
- PCA cluster visualization
- Confusion matrix-style genre view
- Scatter plots
- Cluster profile analysis
- Music recommendation system
